# content.md — industry(業界別 TUNAG ハブページ / 大辞典型)

業界別 LP の入り口となるハブページ。**大辞典型の索引ページ**として位置づける(訪問者が業界一覧から自業界を選んで下層 LP に進む構成)。Hero は CTA を置かず、辞典の表紙のように「タイトル + 短いリード」のみで業界グリッドへ即誘導する。

---


## 3. 業界一覧グリッド(p-industries — 本ページの主役)

**配置意図**: 大辞典の本体。Hero 直下 → Logos → 即グリッド という最短導線で、目的の業界を選べる構成。

- **eyebrow**: `Industries`
- **title**: あなたの業界を選んでください。
- **lead**: 各業界の特徴的な課題と TUNAG の活用パターンを、業界別 LP にまとめています。気になる業界をクリックすると、専用の詳細ページに遷移します。
- **layout_note**: モバイル 1 列 / タブレット 2 列 / デスクトップ 3 列 / ワイド 4 列。下層 LP 未公開業界は `coming-soon` バッジ + クリック無効化。
- **items**(13 業界):
  - `care` 介護 / 多拠点・多職種・夜勤シフト下でも、職員の声を吸い上げる運用へ。 / `available`
  - `manufacturing` 製造 / 工場の現場知を、本社と他拠点へ流通させる仕組みづくり。 / `available`
  - `food` 飲食 / 店舗ごとの空気を見える化し、ブランドの一貫性を保つ。 / `available`
  - `retail` 小売 / 店長層の負荷を下げ、本部との情報流通スピードを上げる。 / `coming-soon`
  - `logistics` 運送 / ドライバーが孤立しない、車内からも参加できるエンゲージメント設計。 / `coming-soon`
  - `warehouse` 倉庫 / 短期離職の連鎖を断つ、入社初日からの定着支援。 / `coming-soon`
  - `construction` 建設 / 協力会社まで含めた現場一体感を、安全文化の土台に。 / `coming-soon`
  - `staffing` 人材派遣 / 派遣スタッフのエンゲージメントを、登録から就業中まで一気通貫で。 / `coming-soon`
  - `car-dealer` カーディーラー / 営業・整備・受付が連携する、店舗ごとの強い組織づくり。 / `coming-soon`
  - `beauty` 美理容 / 個人プレーになりがちなサロン運営を、チームの成長に変える。 / `coming-soon`
  - `hotel` ホテル・旅館 / 季節変動と多国籍スタッフを乗り越える、おもてなしの言語化。 / `coming-soon`
  - `it` IT / リモート前提でも薄まらないカルチャーを、テキストで設計する。 / `coming-soon`

## 4. Solution(TUNAGの普遍価値 — 業界をまたいで効く本質)

- **eyebrow**: `Solution`

- **items**:

## インラインCTA(`p-cta`)挿入の決め方

```yaml
inline_ctas:
  - position: after_solution      # 納得の直後
    title: 自社の業界での具体的な活用イメージをご覧ください
    body: 業界別の専任担当が、貴社の業種・規模・課題に合わせたデモをご用意します。
    primary_cta_text: 業界別デモを依頼する
    ghost_cta_text: 資料をダウンロード
  - position: after_case_studies  # 事例で熱量が上がった直後
    title: 同じ業界の活用事例を、より詳しく
    body: 介護・製造・飲食ほか、業界別の事例集を1冊にまとめてご提供します。
    primary_cta_text: 業界別の事例集をダウンロード
    ghost_cta_text: 個別相談へ
```

ブランドグラデの全面背景使用回数: **Hero + インラインCTA × 2 = 計3回**(design.md §2 配色ルール上限の 4回以内)

## 制作上のメモ(フェーズ2への申し送り)

- **大辞典型**として運用するため、Hero / Final CTA / Contact フォームは持たない。Issues セクションも省略
- Hero はタイトル+リードのみの「表紙」として機能させ、即下のグリッドへ視線を流す
- ページ末尾は **インラインCTA #2 → Footer** の順で締める(Final CTA は無し)
- 「導入実績」表現は使わない。「利用中企業1,400社以上」で統一
- お問い合わせ動線が必要な訪問者向けには、ヘッダー右の「無料デモを見る」ボタンとフッターの「お問い合わせ」リンクを残す(リンク先は外部 / 別ページ想定で `#` プレースホルダ)
- 業界カードの `coming-soon` はクリック無効化(`aria-disabled="true"`)
- 業界一覧グリッドのリンク先は `/industry/<slug>/`(ローカル制作中は `#` で仮置き可)
