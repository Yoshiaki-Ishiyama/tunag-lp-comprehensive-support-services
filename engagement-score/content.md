# content.md — engagement-score

TUNAG のエンゲージメントスコア(β版・行動ログ型サーベイ機能)LP のコピー。
サービス実像:社員が一問も答えない「行動ログ型」サーベイ。TUNAG 上の投稿・コメント・既読・読了データに6つのエンゲージメント軸ごとの係数をかけ、個人・部署単位でリアルタイムにスコアリングする機能。
`design.md §7` 標準構成に沿って章立ては Hero → Logos → Issues → Solution → Strengths → Actions → Final CTA → Contact → Footer。インラインCTAは Solution 後(納得直後)に 1 回のみ。

## 4. Solution(ソリューション概要)

- **eyebrow**: `Solution`
- **title**: 行動ログが、組織を映す。エンゲージメントスコアの3つの柱
- **lead**: TUNAG上の投稿・コメント・既読・読了などの行動ログに、6つのエンゲージメント軸ごとの係数をかけて自動算出。社員に一問も答えさせずに、組織の実像をリアルタイムで捉えます。
- **items**:
  - **label**: `Solution 01`
    **item_title**: 回答負荷ゼロ。行動ログから自動でスコアリング
    **item_text**: 既にTUNAG上で日常的に発生している投稿・コメント・既読データを活用するため、社員への追加の質問は一切発生いたしません。サーベイ疲れとは無縁の運用を実現します。
    **item_image**: `/images/solution/solution_01.jpg`(行動ログ→スコア化のフロー図キャプチャ・height 520px)
  - **label**: `Solution 02`
    **item_title**: 意思の介在なし。本音の組織コンディションをリアルタイムで
    **item_text**: 自己申告に頼らない行動ログベースだから、回答者の建前バイアスが入りません。サイレント退職の兆しも、行動量の変化として早期に検知いたします。
    **item_image**: `/images/solution/solution_02.jpg`(離職予兆アラート画面のキャプチャ)
  - **label**: `Solution 03`
    **item_title**: 6軸×個人・部署で、組織の弱点をピンポイント特定
    **item_text**: 「企業・業務理解度」「理念共感」「文化醸成」「成長意欲」「協働性」「発信性」の6軸で多角化。個人単位・部署単位でスコアを並べ、どこの何が弱いのかが一目で把握できます。
    **item_image**: `/images/solution/solution_03.jpg`(6軸レーダーチャート×セグメントテーブルのキャプチャ)

### 6つのエンゲージメント軸(補足)

Solution 03 で触れた6軸の簡易定義。本文中で軸名を並べる箇所の下に、小さな補足ブロックとして併置する。

- **企業・業務理解度**:自社の事業・業務内容への理解の深さ(社内報・マニュアル等の閲覧に表れる)
- **理念共感**:経営理念や方針への共感度(トップメッセージ・中期経営計画への反応に表れる)
- **文化醸成**:相互承認・称賛文化への参加度(サンクスメッセージ・表彰制度の活用に表れる)
- **成長意欲**:学習・成長機会への取り組み姿勢(研修報告・書籍共有・勉強会参加に表れる)
- **協働性**:他者との連携の質(1on1・自己紹介・メンター制度に表れる)
- **発信性**:自発的な情報発信の量と質(日報・週報・ナレッジ共有に表れる)

### 既存サーベイとの比較(補足テーブル)

`Solution` セクション下部にテーブル or 吹き出しカードとして比較を併置する。デザイン上はシンプルな3列×6行テーブルで構わない。

|                    | パルスサーベイ  | 組織サーベイ   | エンゲージメントスコア |
|--------------------|---------------|-------------|---------------------|
| 頻度               | 週1〜月1回     | 年1〜2回     | リアルタイム/常時     |
| 意思の介在          | 高い(自己申告)| 高い(自己申告)| 低い(行動ログ)      |

| 離職予兆の検知      | △ 遅れがち    | ✗ 年単位で遅延| ◎ 早期検知可能      |
| 部署別比較         | ○ 可能        | ○ 可能       | ◎ 細粒度で可能      |
| 施策効果測定       | △ 主観的      | ✗ タイムラグ大| ◎ 即時フィードバック  |

## 5. Strengths(強み・実績数値のカウントアップ)

- **eyebrow**: `Strengths`
- **title**: 選ばれる3つの理由
- **items**:
  - **metric_number**: `0`
    **metric_unit**: `問`
    **metric_caption**: 社員の回答負荷
    **item_title**: 答えないから、継続できる
    **item_text**: 行動ログから自動算出するため、社員に回答を依頼する必要がありません。サーベイの運用コストも、回答率の低下も発生しません。常時データが蓄積し続けます。
  - **metric_number**: `1`
    **metric_unit**: `プラットフォーム`
    **metric_caption**: 可視化から改善まで一気通貫
    **item_title**: 見つけて、そのまま動ける
    **item_text**: スコアで気づいた課題に対して、1on1の促進、トップメッセージの発信、サンクス・表彰制度の運用など、改善の取り組みをそのままTUNAG上で実行できます。別ツールへの乗り換えが発生しないため、「可視化して終わり」にならず、気づきから打ち手までが一続きで回り続けます。
  - **metric_number**: `365`
    **metric_unit**: `日`
    **metric_caption**: 常時計測(リアルタイム可視化)
    **item_title**: 予兆は、手遅れになる前に掴める
    **item_text**: 行動ログは毎日集積され続けるため、サーベイの回答を待たずに日次でコンディション変化を追跡できます。離職の予兆や、モチベーション低下のサインをリアルタイムに捉え、1on1や面談による早期フォローへとつなげられます。

## 6. Actions(スコアが導くアクション)

- **eyebrow**: `From Signal to Action`
- **title**: スコアが、次の打ち手を教えてくれる
- **lead**: エンゲージメントスコアは、ただの数値では終わりません。「個人」「部署」「全社」の3つのスコープで、今すべき打ち手がアクションとして見えてきます。
- **items**(3スコープの縦積み or 3カラムカード):
  - **scope_label**: `Individual`
    **scope_title**: 個人
    **trigger**: 個人のエンゲージメントスコアが、直近数カ月で大きく下落している
    **action**: 業務過多やモチベーション低下の可能性が考えられるため、**1on1を実施**して状況をヒアリング・フォローする
  - **scope_label**: `Team`
    **scope_title**: 部署
    **trigger**: 特定の部署のみ、スコアが顕著に低い
    **action**: マネジメント上の課題が疑われるため、該当部署の**マネージャーへのヒアリング**を実施し、1on1・キャリア面談の運用を見直す
  - **scope_label**: `Company`
    **scope_title**: 全社
    **trigger**: 全社的に「理念共感」の軸が低下している
    **action**: **経営・役員から事業の方向性や会社の未来について発信**する機会を増やし、トップメッセージの頻度・内容を強化する

## 付録: サービス事実メモ(ライティング参考)

PDFから抽出したライティングの根拠。コピー編集時はこの事実を動かさないこと。

- **TUNAGの位置づけ**: 人と組織のつながりを作り、信頼関係を育む「エンゲージメント経営プラットフォーム」

- **現状**: β版で先行提供中。今後TUNAGダッシュボードへの反映を予定
- **会社概要**: 株式会社スタメン(東証グロース:4019) / 2016年創業 / 2017年TUNAGリリース / 2020年「働きがいのある会社ランキング」第1位 / デロイトトーマツFast50 第1位(以降6年連続受賞) / 現在のTUNAG利用企業数1,400社以上(※累計導入社数は非公開) / 連結180名(2025年9月末時点)
